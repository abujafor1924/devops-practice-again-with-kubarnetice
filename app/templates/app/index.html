CI/CD Pipeline Overview & Deployment Status

1. CI: Run Unit Tests

Executes Django unit tests automatically on a Python matrix (3.11 & 3.12). Ensures code integrity before build.

2. Build & Push

Builds optimized Docker image and tags it with commit SHA, then pushes it to Docker Hub repository automatically.

3. CD: Auto Deployment

GitHub Action SSHs into deployment target, pulls the fresh image, runs migrations, and updates the application live.
